# mcp-nlp-bench

Исследование эффективности комбинированного подхода **LLM + классические алгоритмы компьютерной лингвистики** через Model Context Protocol.

## Суть

Сравниваем два подхода к решению NLP-задач для русского языка:

- **LLM-only** — языковая модель решает задачу самостоятельно
- **LLM + MCP** — языковая модель вызывает классические NLP-алгоритмы через MCP-протокол

Задачи: NER, POS-теггинг, лемматизация.
